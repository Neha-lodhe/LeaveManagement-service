# Leave Management System

A Spring Boot REST API application for managing employees, leave requests, and leave balances.

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven
- Postman
- Git & GitHub

## Features

### Employee Management
- Add Employee
- View Employee
- Delete Employee

### Leave Request Management
- Apply Leave
- Approve Leave
- Reject Leave

### Leave Balance Management
- Track available leaves
- Update leave balance automatically

## API Endpoints

### Employee APIs

POST /employees

GET /employees

GET /employees/{id}

DELETE /employees/{id}

## Database

PostgreSQL
